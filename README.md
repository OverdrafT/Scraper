# Scraper
This is backend service for scraping data for predefined tickers from finance.yahoo.com
## Overview
Service receives ticker's list as input and returns ticker's data scraped from finance.yahoo.com 
## Prerequisites
- Python 3.8.6
- Requests lib
- BeautifulSoap lib
## Getting started
- Clone repository
- `python scraper.py`

Enjoy!
